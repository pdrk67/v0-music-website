O que é TypeScript?

TypeScript é uma linguagem de programação construída sobre JavaScript que adiciona tipos estáticos. Isso significa que você pode especificar qual tipo de dados uma variável deve ter (número, texto, etc).

Por que usar TypeScript?

1. Detecta erros ANTES de executar o código
2. Torna o código mais legível e fácil de entender
3. Facilita trabalhar em equipes grandes
4. Muitas empresas usam profissionalmente

Exemplo Simples:

const nome: string = "João";
const idade: number = 25;
const estudante: boolean = true;

Carreira em TypeScript:

- Desenvolvedor Full-Stack
- Desenvolvedor Frontend
- Desenvolvedor Backend (Node.js)
